AC3006 Enterprise-Grade Wireless Controller

The AC3006 is an enterprise-grade multi-service wireless controller for next-generation simplified networks. It supports firewall, wireless controller (AC) and routing functionalities, making it suitable for high-capacity, high-reliability, and multi-service scenarios. The AC3006 can manage up to 256 wireless APs (local forwarding) and features standardization, advanced architecture, multi-service support, high reliability, ease of maintenance, and cost-effectiveness. It works with APs to deliver wireless coverage for medium-scale enterprise campuses and office environments.

Product Features

Multi-Scenario Networking Adaptability

  • Wi-Fi 7 Support: AC3006 supports IEEE 802.11be protocol, managing Wi-Fi 7 APs and terminals while maintaining backward compatibility with legacy APs (802.11a/b/g/n/ac/ax)
  • Flexible Topology: Supports Layer 2/Layer 3 and WAN-based AP-AC networking
  • Traffic Forwarding Modes:
    • Centralized forwarding (traffic processed by AC)
    • Local forwarding (traffic processed at AP)
    • DHCP handling flexibility (local or AC-based)
  • Seamless Roaming: Supports Layer 2/3 roaming across VLANs even in local forwarding mode
  • Dual-WAN Load Balancing: Two WAN ports for bandwidth aggregation and failover (auto-switch on link failure)
  • Built-in Authentication Services: Integrated RADIUS/Portal Server simplifies WPA2-Enterprise and web authentication without external servers

AI-Driven Wireless Optimization

  • Auto Channel/Power Adjustment: Dynamically selects optimal channels and adjusts AP transmit power to reduce interference
  • Band Steering: Prioritizes 5GHz for dual-band clients to alleviate 2.4GHz congestion
  • Heatmap Visualization: Displays signal strength, SNR, and coverage gaps for AP placement optimization
  • Spectrum Analysis: Detects non-Wi-Fi interference (e.g., Bluetooth, microwaves) and triggers self-healing channel switching

Advanced Security Protocols

  • WPA3 : SAE authentication, GCMP-256 encryption, and 192-bit enterprise-grade security.
  • DingTalk/WeChat Mini Program Auth: Integrates with DingTalk accounts or WeChat Mini Programs for secure access control.
  • Portal: Supports multi-vendor Portal servers for SMS, LDAP, and password-based authentication.
  • Network Isolation: Separates internal (office) and guest traffic via multi-SSID and VLANs.
  • Layer 7 DPI:Application identification based on Layer 7 signature database, with policy-driven blocking of high-risk applications (e.g., Tor, cryptocurrency mining software), prioritizing critical services (e.g., video conferencing/SAP).

High Availability

  • 1:1 VRRP Hot Backup: Sub-second failover for zero service disruption.
  • N+1 Backup:Shared backup AC for cost-efficient redundancy.
  • Flexible Licensing:License sharing across primary/backup ACs and virtualized instances.
  • Anti-Attack Features: Flooding attack detection, DTLS encryption for CAPWAP tunnels.

Network Management

  • GUI & Topology Visualization: Real-time AP load, client distribution, and interference heatmap monitoring.
  • Northbound APIs:RESTful API and SNMPv1/v2/v3 for automation and third-party integration.
  • Cloud Management:Centralized control via cloud platforms, reducing maintenance complexity.
